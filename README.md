## Available languages
- Hindi
- English
- Bengali
- Punjabi
- Tamil
- Telgue
- Malayalam

## How to add a new language 
    
Follow these steps
- Open `/src/js/ui/translation/variables.js`, edit each variable with equivalents in your langauge
- That's it, make a PR
